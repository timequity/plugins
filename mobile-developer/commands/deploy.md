# /mobile:deploy

Deploy using app-store-deploy skill.

## Process

1. **Build** - EAS Build for both platforms
2. **Test** - TestFlight / Internal track
3. **Submit** - App Store Connect / Play Console
4. **Monitor** - Review status

## Prerequisites

- Apple Developer Account
- Google Play Developer Account
- EAS credentials configured
