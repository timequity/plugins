# /mobile:create

Create a new mobile app using expo-workflows skill.

## Input Required

1. **App name** - Display name for app stores
2. **Bundle ID** - com.company.appname
3. **Features** - Auth, payments, push, etc.

## Output

- Expo project with TypeScript
- Navigation setup
- Basic screens
- Store configuration
