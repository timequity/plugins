# Mobile Developer

Create cross-platform mobile applications with modern tooling.

## Skills

| Skill | When to Use |
|-------|-------------|
| **react-native-expert** | Core RN patterns, navigation, performance |
| **expo-workflows** | EAS Build, updates, managed workflow |
| **mobile-ui** | Gestures, animations, platform-specific UI |
| **app-store-deploy** | iOS App Store, Google Play submission |
| **offline-first** | Local storage, sync, conflict resolution |
| **push-notifications** | Firebase, APNs configuration |
| **deep-linking** | Universal links, app links |

## Commands

| Command | Purpose |
|---------|---------|
| `/mobile:create` | Scaffold new mobile app |
| `/mobile:screen` | Create screen with navigation |
| `/mobile:deploy` | Build and deploy to stores |
| `/mobile:test` | Run mobile tests |

## Templates

- **expo-typescript** - Expo + TypeScript + navigation
- **mobile-saas** - Full app with auth, payments, push

## Philosophy

- **Cross-platform first** - Share code, respect platforms
- **Performance matters** - 60 FPS, fast startup
- **Offline capable** - Works without network
- **Store ready** - Meet all requirements
