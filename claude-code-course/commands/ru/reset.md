# /course:reset — Сброс прогресса

Удаляет весь прогресс и начинает курс заново.

## Синтаксис

```
/course:reset
```

## Алгоритм

### 1. Запросить подтверждение

```
Ты уверен, что хочешь сбросить прогресс?

Это удалит:
- Все завершённые уроки
- Статистику попыток
- Дату начала курса

Напиши "да" для подтверждения или "нет" для отмены.
```

### 2. При подтверждении

```bash
rm -f "$HOME/.claude-course/progress.json"
```

И показать:
```
Прогресс сброшен.

Готов начать заново? Введи `/course`
```

### 3. При отмене

```
Сброс отменён. Твой прогресс сохранён.

Текущий урок: {N}
Продолжить: `/course`
```

## Важно

- Всегда спрашивать подтверждение
- Не удалять автоматически
- Показать что именно будет удалено
