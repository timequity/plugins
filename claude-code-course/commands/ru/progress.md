# /course progress — Детальный прогресс

Показывает подробную статистику прохождения курса.

## Синтаксис

```
/course progress
```

## Алгоритм

### 1. Загрузить прогресс

Используй скилл `progress-manager`:
```bash
cat "$HOME/.claude-course/progress.json"
```

### 2. Отобразить детальную статистику

```
# Твой прогресс в курсе Claude Code

## Общий прогресс

████████░░░░░░░░░░░░ 40% (2/5 уроков)

Начат: 15 января 2025
Времени потрачено: ~45 минут

## Уроки

### Урок 1: Основы Claude Code
Статус: ✓ Завершён
Начат: 15 янв, 10:00
Завершён: 15 янв, 10:15
Практика: 1 попытка

### Урок 2: Твой первый скилл
Статус: ● В процессе
Начат: 15 янв, 10:20
Теория: ✓ прочитана
Практика: ○ не завершена (2 попытки)

### Урок 3-5
Статус: ○ Не начаты

## Что дальше?

Продолжи урок 2 — осталась практика.
`/course practice` — выполнить задание
```

### 3. Расчёт времени

Приблизительное время:
- `started_at` первого урока до `completed_at` последнего завершённого
- Или до текущего момента если курс не завершён

### 4. Если прогресса нет

```
# Прогресс

Ты ещё не начал курс.

Введи `/course` чтобы начать обучение!
```

### 5. Если курс завершён

```
# Поздравляем!

████████████████████ 100% — Курс завершён!

Ты освоил:
- Основы Claude Code
- Создание скиллов
- Работу с агентами
- Хуки и автоматизацию
- Построение приложений

Начат: 15 января 2025
Завершён: 15 января 2025
Всего времени: ~2 часа

Что дальше?
- Создай свой первый плагин
- Изучи marketplace
- Помоги другим — поделись опытом
```

## Формат дат

- Полная: "15 января 2025, 10:30"
- Короткая: "15 янв"
- Относительная: "сегодня", "вчера"
