# /lesson N — Переход к конкретному уроку

Открывает урок по номеру.

## Синтаксис

```
/course lesson 1
/course lesson 2
...
/course lesson 5
```

## Алгоритм

### 1. Валидация номера

Проверь что N от 1 до 5. Если нет:
```
Урок {N} не существует. Доступны уроки 1-5.
```

### 2. Загрузка контента урока

Контент уроков находится в:
```
plugins/claude-code-course/lessons/
├── lesson-1.md
├── lesson-2.md
├── lesson-3.md
├── lesson-4.md
└── lesson-5.md
```

Прочитай файл `lesson-{N}.md` и покажи содержимое.

### 3. Обновление прогресса

1. Установи `current_lesson: N`
2. Если нет записи для урока N, создай:
   ```json
   "N": {
     "started_at": "ISO_DATE",
     "theory_read": false,
     "practice_completed": false,
     "practice_attempts": 0
   }
   ```
3. Сохрани прогресс

### 4. Формат урока

После показа теории:
```
---

Теория изучена! Теперь практика.

Введи `/course practice` чтобы начать практическое задание.
```

И обнови `theory_read: true`.

## Структура урока

Каждый урок содержит:
1. **Заголовок** — название урока
2. **Цели** — что узнаешь
3. **Теория** — основные концепции (короткие блоки)
4. **Примеры** — код или команды
5. **Резюме** — ключевые моменты

## Тон

- Обращайся на "ты"
- Объясняй просто, без жаргона
- Показывай реальные примеры
- После теории — сразу к практике
