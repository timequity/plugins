# /course:list — Список уроков

Показывает все уроки со статусами прохождения.

## Синтаксис

```
/course:list
```

## Алгоритм

### 1. Загрузить прогресс

```bash
cat "$HOME/.claude-course/progress.json" 2>/dev/null || echo '{}'
```

### 2. Показать таблицу уроков

```
# Уроки курса Claude Code

| # | Урок | Время | Статус |
|---|------|-------|--------|
| 1 | Основы Claude Code | ~15 мин | ✓ |
| 2 | Твой первый скилл | ~20 мин | ● |
| 3 | Агенты-помощники | ~25 мин | ○ |
| 4 | Хуки и автоматизация | ~20 мин | ○ |
| 5 | Собираем приложение | ~30 мин | ○ |

**Легенда:** ✓ завершён • ● в процессе • ○ не начат

Текущий: Урок 2
Команда: `/course lesson 2` для продолжения
```

### 3. Статусы

| Символ | Условие |
|--------|---------|
| ✓ | `practice_completed: true` |
| ● | `started_at` есть, но `practice_completed: false` |
| ○ | Нет записи в `lessons` |
