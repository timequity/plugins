## Шаг 2: Скилл для стилей

Создай скилл который знает стиль твоего приложения:

```markdown
---
name: todo-style
description: |
  Стили для Todo приложения.
  Use when: нужно добавить или изменить стили.
  Triggers: "стили", "css", "дизайн".
---

# Todo App Styles

## Цветовая схема
- Фон: #f5f5f5
- Карточки: #ffffff
- Акцент: #4CAF50
- Текст: #333333

## Правила
- Минималистичный дизайн
- Закруглённые углы (8px)
- Тени для карточек
- Анимации при hover
```

---

## Шаг 3: Агент для логики

Создай агента который пишет JavaScript:

```markdown
---
name: todo-logic
description: Реализует логику Todo приложения.
tools:
  - Read
  - Write
  - Edit
model: sonnet
---

# Todo Logic Agent

Реализуй функционал Todo списка.

## Требования

1. Хранение в localStorage
2. Функции: addTodo, toggleTodo, deleteTodo
3. Рендер списка при изменениях
4. Никаких внешних зависимостей

## Структура данных

```js
const todos = [
  { id: 1, text: "Задача", done: false }
]
```
```

## Шаг 4: Хук для форматирования

Добавь в `.claude/settings.json`:

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit",
        "hooks": [
          {
            "type": "command",
            "command": "prettier --write \"$CLAUDE_FILE_PATH\" 2>/dev/null || true"
          }
        ]
      }
    ]
  }
}
```

## Шаг 5: Деплой

Варианты:
1. **GitHub Pages** — бесплатный хостинг для статики
2. **Vercel** — один клик деплой
3. **Netlify** — аналог Vercel

**Для GitHub Pages:**

```bash
# Создай репозиторий
gh repo create todo-app --public

# Запуши код
git add .
git commit -m "Initial commit"
git push -u origin main

# Включи Pages
gh repo edit --enable-wiki=false
# Затем: Settings → Pages → Source: main branch
```
