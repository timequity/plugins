## Что такое Claude Code?

Claude Code — это CLI-инструмент, который превращает Claude в твоего напарника по программированию. Ты общаешься с ним в терминале, а он помогает писать код, искать файлы, запускать команды.

**Главное отличие от ChatGPT/веб-версии Claude:**
- Claude Code работает в ТВОЁМ терминале
- Видит ТВОИ файлы
- Может ВЫПОЛНЯТЬ команды (с твоего разрешения)

---

## Интерфейс

Когда ты запускаешь `claude`, видишь:

```
╭─────────────────────────────────────╮
│  Claude Code                        │
│  /help for commands                 │
╰─────────────────────────────────────╯

>
```

`>` — сюда пишешь свои сообщения.

---

## Основные команды

| Команда | Что делает |
|---------|------------|
| `/help` | Показывает справку |
| `/clear` | Очищает историю диалога |
| `/compact` | Сжимает контекст (если разговор длинный) |
| `/cost` | Показывает потраченные токены |
| `Ctrl+C` | Прерывает текущее действие |
| `Esc` → `Esc` | Выход из Claude Code |

Попробуй ввести `/help` прямо сейчас!

## Как эффективно общаться с Claude

### Правило 1: Будь конкретным

```
❌ Плохо: "Сделай красиво"
✓ Хорошо: "Добавь отступы 16px между секциями"
```

### Правило 2: Давай контекст

```
❌ Плохо: "Исправь баг"
✓ Хорошо: "Функция login() возвращает null вместо user объекта"
```

### Правило 3: Разбивай большие задачи

```
❌ Плохо: "Создай интернет-магазин"
✓ Хорошо: "Создай модель Product с полями name, price, description"
```

### Правило 4: Проверяй результат

Claude может ошибаться. Всегда проверяй что он сделал:
- Читай сгенерированный код
- Запускай тесты
- Проверяй логику

## Что Claude может делать

| Можно | Нельзя |
|-------|--------|
| Читать твои файлы | Читать чужие компьютеры |
| Запускать команды | Запускать без разрешения |
| Искать по коду | Подключаться к интернету* |
| Создавать файлы | Удалять системные файлы |

*Claude может использовать web search если включён.

## Практика

Теперь твоя очередь!

**Задание:** Создай файл `hello.txt` в текущей директории с текстом "Привет от Claude!"

**Как сделать:**
1. Попроси Claude создать файл
2. Проверь что файл создан (`ls` или `cat hello.txt`)

**Подсказка:** Просто напиши "Создай файл hello.txt с текстом Привет от Claude!"

Когда закончишь, скажи "готово" и я проверю результат.

---

## Проверка практики

Когда пользователь говорит "готово", выполни проверку:

```bash
# Проверка урока 1
if [ -f "hello.txt" ] && grep -q "Привет" hello.txt; then
  echo "✓ Практика выполнена!"
else
  echo "✗ Файл hello.txt не найден или не содержит приветствие"
fi
```

**При успехе:**
- Обнови прогресс: `practice_completed: true`

**При неудаче:**
