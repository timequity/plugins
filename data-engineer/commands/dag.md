# /data:dag

Create an Airflow DAG using airflow-workflows skill.

## Input Required

1. **DAG name** - Descriptive identifier
2. **Schedule** - Cron expression or preset
3. **Tasks** - What operations to perform?
4. **Dependencies** - Task ordering

## Output

- Python DAG file
- Operator configuration
- Error handling setup
