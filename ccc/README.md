# Claude Code Course (CCC)

Master Claude Code in 5 hands-on lessons — from basics to building applications.

## What You'll Learn

| Lesson | Topic | Duration |
|--------|-------|----------|
| 1 | Claude Code Basics | ~15 min |
| 2 | Your First Skill | ~20 min |
| 3 | Agent Helpers | ~25 min |
| 4 | Hooks and Automation | ~20 min |
| 5 | Building an Application | ~30 min |

## Installation

### Option 1: Download ZIP (Recommended)

1. Go to [Releases](../../releases)
2. Download the latest `ccc.zip`
3. Extract to `~/.claude/plugins/ccc/`

```bash
# macOS/Linux
mkdir -p ~/.claude/plugins
cd ~/.claude/plugins
unzip ~/Downloads/ccc.zip -d ccc
```

### Option 2: Git Clone

```bash
cd ~/.claude/plugins
git clone https://github.com/timequity/ccc
```

### Verify Installation

1. Restart Claude Code
2. Type `/ccc:course` to start the course

## Usage

| Command | Description |
|---------|-------------|
| `/ccc:course` | Start or continue the course |
| `/ccc:lesson 1` | Go to specific lesson |
| `/ccc:practice` | Practice current lesson |
| `/ccc:progress` | Show your progress |
| `/ccc:reset` | Reset and start over |

## Features

- **Interactive lessons** with theory and practice
- **Progress tracking** — continue where you left off
- **Tutor agent** — ask questions anytime
- **Graduation app** — build a real application in lesson 5

## Requirements

- Claude Code CLI installed
- Basic terminal knowledge
