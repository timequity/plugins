# Deploy

Загрузи скилл `fullstack-developer:devops` и помоги с деплоем.

1. Проверь наличие Dockerfile и CI/CD конфигов
2. Если нет — предложи создать на основе assets из скилла
3. Проверь готовность к деплою (тесты, билд)
4. Помоги с настройкой деплоя на выбранную платформу

Аргументы: $ARGUMENTS
