# Code Review

Загрузи скилл `fullstack-developer:code-review` и выполни code review текущих изменений.

1. Получи контекст через `git diff` и `git log`
2. Проанализируй изменения по чеклисту из скилла
3. Выдели Critical, Important и Minor issues
4. Предложи конкретные улучшения

Аргументы: $ARGUMENTS
